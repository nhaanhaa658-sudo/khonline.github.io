Setup Git
បើក Terminal / CMD → វាយ command ខាងក្រោម
git config --global user.name "Your Name"
git config --global user.email "you@email.com"
៣
Clone Repository
Copy link ពី GitHub → Clone មកកុំព្យូទ័រ
git clone https://github.com/username/cv-service.git
cd cv-service
